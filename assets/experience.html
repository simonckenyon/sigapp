Simon is CTO of a start-up which is developing innovative mobile and cloud based software for the Oil and Gas industry. He leads a team who are using Agile methods to rapidly create applications for the E&P sector of this industry.

Clients run on Android and Apple devices and communicate with a server side that is a web system written using Groovy of Grails and MySQL. Communication between the clients and servers utilise RESTful web services. The servers run in Rackspace.
